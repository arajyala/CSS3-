Cascading style sheet(CSS)

    The major drwbacks of Html output are
  1. All the text contents of html will appear black in colour on the Browser
  2. All the text contents will appear always to the left side of the browser
  3. To overcome the above drawbacks HTML must be Combined with CSS

Note:CSS is basically preferred to enhance the Html output on the browser.

To Write CSS code we have 3 Ways

  • Inline CSS
  • Internal CSS
  • External CSS

Inline CSS: In inline Css, all the CSS code is given within HTML tag by Using "Style Attribute"

<h1> Style="color:red">Welecome to CSS </h1>;

Internal CSS: In Internal Css, all the CSS code is given within HTML file by Using "Style tag"

<Style> h1{color:red} </style> <h1>Welcome to CSS!!</h1>

External CSS: In External Css, all the CSS code is must be given in seaprate CSS file and link it to the html file by using link tag

< link rel="stylesheet" href="filename.css" />

Note:In case all the 3 types of codes is used together for 1 tag priority first is given to inline CSS. Followed by internal CSS and at last it goes to external CSS.
