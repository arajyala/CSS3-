CSS Selectors

Element Selector

The element selector selects HTML elements based on the element name.

Example

p{ text-align:center; color:red; }

Id Selector

The id selector uses the id attribute of an HTML element to select a specific element.

The id of an element is unique within a page, so the id selector is used to select one unique element!

To select an element within a specific id, write a hash (#) charecter, followed by the id of the element./p>

Example

#p1{ text-align:center; color:red; }

Class Selector

The Class selector selects HTML elements with a specific class attribute.

To select elements with a specific class, write a period(.) charecter, followed by the class name.

Example

.center{ text-align:center; color:red; }

Note:

  • Id will have more priority then class Selector.
  • Class selector will have more priority then element selector.
  • Inline css will have more priority then Id and Class Selectors
